What this project does

The page includes two fully working analytics workflows. You can use the built-in sample datasets or upload your own CSV files. All calculations run live in JavaScript so the deployed page remains functional on GitHub Pages.

Salary Prediction Audit

Removes duplicates, fills nulls with medians, filters unrealistic rows, removes outliers, trains a regression model, and predicts salary for a candidate with 15 years of experience.

Customer Segmentation Dashboard

Scales features, applies the elbow method, trains K-Means clustering, computes silhouette score, and assigns a new customer into a loyalty segment.
